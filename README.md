# SABAP2PentadList
Takes a pentad code as input and generates a checklist of potential species

Run locally with `XENO_CANTO_API_KEY=your-key ntl dev`

The bird call lookup now uses xeno-canto API v3, which requires an API key.
You can use either `XENO_CANTO_API_KEY` or `XC_API_KEY` locally and in Netlify.
